# Camera Rental Management System (C++)

A console-based camera rental management system built using C++, designed to simulate real-world rental workflows and apply fundamental data structure concepts.

## Overview

This project was developed as part of a Data Structures (Struktur Data) course. It focuses on managing camera inventory and rental transactions through a structured and logical approach using C++.

The system allows users to perform basic rental operations such as viewing available items, renting cameras, and returning them, while maintaining organized data handling.

## Features

* Manage camera inventory
* Search for available cameras
* Rent and return cameras
* Display and organize data
* Sorting functionality for better data arrangement

## Tech Stack

* **Language:** C++
* **Concepts Applied:**

  * Arrays for data storage
  * Sorting algorithms for organizing data
  * Searching techniques for item lookup
  * Structured programming and control flow

## System Workflow

1. User views available camera list
2. User selects a camera to rent
3. System processes rental transaction
4. User can return the camera
5. System updates inventory accordingly

## Learning Objectives
* Apply data structures in a real-case scenario
* Understand how to manage and organize structured data
* Implement sorting and searching algorithms
* Build logical and efficient program flow in C++
